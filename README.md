# Maintenance Bot
A bot which can hold other bots on Maintenance.

## Variables 
```
TOKEN - Your telegram bot token from @BotFather.
OWNER_ID - Your Telegram User ID.
MONGODB_URL - Get a db url from mongodb.com.
```
